# RISC-V-Processor-using-Pipeline-Architecture
Pipelined MIPS-inspired processor implemented in Verilog HDL featuring a 5-stage pipeline (IF, ID, EX, MEM, WB), dual-clock architecture, hazard forwarding unit, and support for ADD, ADDI, MUL, BEQ instructions. Computes integer square roots algorithmically using iterative multiplication loops.
